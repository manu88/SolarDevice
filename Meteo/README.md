# Meteo

Collecte les données météo depuis [infoclimat](http://www.infoclimat.fr) ainsi que les [éphémérides](https://api.sunrise-sunset.org) et publie des messages `osc`.

## usage

```bash
#important: doit être lancé depuis le répertoire racine du repo, i.e, le répertoire parent.
python3 -m Meteo.main config/config-test.json
```
